Python Старт

Перші кроки в програмуванні через ігри та малюнки

01

Знайомство з Python

Встановлення, IDLE, print(), перша програма "Hello, World!"

02

Змінні та типи даних

Числа, рядки, True/False. Математика в коді.

03

Умови if / else

Вчимо Python "думати": порівняння та розгалуження логіки.

04

Цикли for та while

Автоматизація повторень. Будуємо патерни та лічильники.

05

Функції

Пишемо власні команди: def, return, параметри.

06

Списки

Зберігання наборів даних, індекси, len(), sorted().

07

Turtle: малюємо кодом

Графіка, фігури, кольори, спіралі та зірки.

08

Minecraft + Python (mcpi)

Підключення до Minecraft Education Edition, будуємо споруди та лабіринти кодом.

09

Знайомство зі штучним інтелектом

Що таке ШІ та машинне навчання — пояснення на прикладах, гра "навчи комп'ютер розрізняти котів та собак".

Python: Ігри та боти

Від синтаксису до власної гри та Telegram-бота

01

Основи та VS Code

Середовище розробника, синтаксис, типи, умови, цикли.

02

Структури даних

Списки, словники, кортежи, set. Коли що використовувати.

03

Функції та модулі

Scope, рекурсія, import, власні модулі.

04

Файли та обробка помилок

Читання/запис файлів, try/except, власні виключення.

05

Робота з API

requests, JSON, отримання даних з відкритих API (погода, курси валют).

06

Pygame: основи

Вікно, події, малювання фігур, таймер, FPS-контроль.

07

Pygame: спрайти та анімація

Завантаження зображень, спрайт-листи, плавний рух, ефекти.

08

Pygame: повноцінна гра

Колізії, рахунок, рівні, звуки, меню старту та кінця гри.

Telegram-бот: основи

aiogram, команди, клавіатури, відповіді на повідомлення.

10

Telegram-бот: стани та логіка

FSM (машина станів), сценарії діалогу, зберігання даних користувача.

11

ШІ у повсякденних задачах

Підключення до OpenAI API, генерація тексту та зображень із Python-скрипту.

Python Pro: до портфоліо

Глибокий Python, автоматизація та реальні продукти

01

Python Advanced: генератори та декоратори

lambda, map/filter, генератори, декоратори — пишемо елегантний код.

02

ООП: класи та об'єкти

Класи, методи, наслідування, інкапсуляція, dunder-методи.

03

Бібліотеки та pip

requests, json, datetime, virtual environment — робота з зовнішніми пакетами.

04

Парсинг даних

BeautifulSoup, requests-html — автоматичний збір інформації з сайтів.

05

Бази даних: SQLite

SQL: SELECT, INSERT, UPDATE, DELETE. Підключення через sqlite3 та SQLAlchemy.

06

Telegram-бот з БД

aiogram + SQLite — реєстрація, профілі, адмін-панель, FSM-діалоги.

07

Веб-API на Flask

Роутінг, GET/POST, шаблони Jinja2, JSON API, автентифікація.

08

Автоматизація та аналіз даних

pandas, openpyxl — читання Excel, статистика, автогенерація звітів.

09

Основи Machine Learning

scikit-learn, numpy, matplotlib — лінійна регресія, класифікатори, візуалізація.

10

Нейронні мережі на практиці

TensorFlow/Keras — будуємо просту нейромережу, розпізнавання рукописних цифр (MNIST).

11

Робота з LLM та OpenAI API

Підключення ChatGPT до власного додатку, prompt engineering, RAG-системи.

12

Git, GitHub та деплой

Гілки, pull request, деплой Flask на Railway/Render, змінні оточення.

13

Тестування та якість коду

pytest, unittest, mock, linting (flake8/black) — код як у реальному проекті.
