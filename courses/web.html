Чого навчиться підліток

8 модулів — від першого тегу HTML до публікації сайту-портфоліо

📄

HTML5 — структура веб-сторінки

Теги, атрибути, семантична розмітка. Створення першої сторінки із заголовком, параграфами, посиланнями та зображеннями.

🎨

CSS3 — стилі та краса

Кольори, шрифти, відступи, рамки. Робота з box-model. Стилізація першої сторінки.

📐

Flexbox та Grid — сучасна верстка

Створення гнучких та сіткових макетів. Адаптивність без Bootstrap — чистий CSS.

📱

Адаптивний дизайн (Responsive)

Media queries, мобільна версія, метатег viewport. Сайт однаково добре виглядає на телефоні та комп'ютері.

⚡

JavaScript — інтерактивність

Змінні, функції, події. Кнопки, спливаючі вікна, зміна теми — сайт оживає!

🗂️

DOM — керування сторінкою

Як JS змінює HTML «на льоту»: показ/приховування елементів, додавання контенту, лічильники.

🌍

Хостинг та публікація сайту

Git, GitHub Pages або Netlify — публікуємо сайт в інтернет безкоштовно. Налаштування домену.

🏆

Фінальний проект — сайт-портфоліо

Багатосторінковий адаптивний сайт з розділами «Про мене», «Проекти» та контактною формою — готове портфоліо!
